# Finder

Finder is a simple search engine with an AI component called Find4U. You can try it out [here](https://finder-locw.onrender.com).

See https://finder-locw.onrender.com/docs/dev/guide.html for instructions on how to run Finder locally, deploy it, etc. If it's taking too long to load and you're getting antsy, look at [the page's source](docs/source/dev/guide.md).
