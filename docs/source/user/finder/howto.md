# Using Finder

For a general overview of Finder, see [the introduction](index.md). This guide assumes you're familiar with basic UI elements such as search boxes, links, and navigation components.

## Main page

![Finder UI](_static/interface.png)

Pictured above is Finder's main page. Below the colorful "Finder!" text is a searchbox which you can use to search. Below that are links. The first link is a link to the full list of pages indexed by Finder. As of the time of writing (2026-06-20) this includes 1,673 pages on the official Render deployment. The second link is a link to Find4U. For more info on that, see [here](../find4u/index). The third link links to the docs you're reading right now.

## Interlude: What are "docs"?

Documentation (docs) is fancy developer jargon for "help and support page" except without contacting any customer support person.

## Searching

After you type something in a searchbox and press the little magnifying glass, you will be taken to a search results page with results for your search.
