Welcome to Finder's internal change log—the commit log. We decided against writing a blog or a changelog by hand because frankly, that would be exhausting. Instead, this page pulls straight from the source: every commit made to Finder’s codebase, in chronological honesty.

Finder uses Git for version control, and this interface is powered by the GitPython library. That means the history here is real—no curation, no sugar-coating. If something broke and got frantically patched five minutes later, you’ll see it. If the `.gitignore` mysteriously ate itself, yep, you’ll see that too.

For commit-by-commit details, click into any entry. You’ll find diffs, descriptions, and the occasional quiet scream of a developer preserved in the commit message.
